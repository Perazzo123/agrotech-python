# Sistema de Irrigação Inteligente

## Descrição
Sistema desenvolvido em Python para auxiliar no controle de irrigação em plantações.

## Problema
A irrigação muitas vezes é feita sem critério, gerando desperdício de água.

## Solução
O sistema analisa:
- Umidade do solo
- pH
- Níveis de nutrientes

E decide automaticamente se a irrigação deve ser ativada.

## Regra
Irrigação é ligada quando:
- Umidade < 50%
- pH entre 5 e 7
- Pelo menos 2 nutrientes OK

## Tecnologias
- Python
- JSON
